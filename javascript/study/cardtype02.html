사이트 만들기 : 카드 유형 02

사이트를 만들기에 앞서 다양한 유형으로 나누어서 공부해보려고 합니다.
가장 먼저 공부해 볼 유형은 카드유형 입니다.
피그마로 먼저 레이아웃을 잡아준 뒤에 수치대로 코딩합니다.
이번 카드 만들기에는 svg파일 삽입, flex가 추가됩니다.

01. Figma로 레이아웃 잡기

Figma로 전체적인 틀을 잡아줍니다.

03. CSS

Figma에서 정해놓은 수치대로 CSS를 넣어 줍니다.
flex를 이용하여 4개씩 2층 구조를 잡아줍니다.

{
    /* fonts */
    @import url('https://webfontworld.github.io/gmarket/GmarketSans.css');

    .gmarket {
        font-family: 'GmarketSans';
        font-weight: 400;
    }
    /* reset */
    * {
    margin: 0;
    padding: 0;
    }
    a {
        text-decoration: none;
        color: #000;
    }
    img {
        width: 100%;
    }
    h1, h2, h3, h4, h5, h6 {
        font-weight: normal;
    }
    /* common */
    .container {
        width: 1160px;
        padding: 0 20px;
        margin: 0 auto;
        min-width: 1160px;
    }
    .section {
        padding: 120px 0;
    }
    .section > h2 {
        font-size: 50px;
        line-height: 1;

    .section > p {
        font-size: 22px;

    }
